Batch redaction

Upload a folder of images that contain personal information. Each image is redacted on this device by the same vision pipeline that protects live Holo sessions, and you get the cleaned set back as a zip, ready to use as training data.

Drop a folder here, or click to choose one PNG, JPEG, and WebP up to 16 MB each. Files never leave 127.0.0.1. Or pick individual files
Redaction runs entirely on-device. Nothing is stored server-side: originals are discarded after each image is processed, and only the painted copies come back to this page.
